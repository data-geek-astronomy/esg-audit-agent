# ESG Carbon Footprint Audit Agent

An AI-powered ESG auditing agent that calculates Scope 1, 2, and 3 greenhouse gas emissions using GHG Protocol methodology and generates CDP-style disclosure reports.

## Features
- GHG Protocol-aligned emissions calculation (Scope 1/2/3)
- Carbon intensity benchmarking
- Top reduction opportunity identification
- CDP disclosure paragraph generation
- Carbon offset recommendations
- Powered by n8n automation + OpenAI GPT-4
